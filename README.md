# PyCodeFlow: Reflective Execution Harness

A framework for transforming static codebases into self-reflective, event-driven organisms with causal lineage tracking.

## Overview
PyCodeFlow instruments static codebases via AST analysis and maintains a biographical ledger of all structural interactions.

## Quick Start
1. Install dependencies (standard requirements).
2. Run a scan: `python3 main.py scan <target_file>`
3. Query history: `python3 main.py reflect TRACE <target_file>`

## Methodology
PyCodeFlow employs an "Additive Wrapper Pattern," decoupling code perception from event storage to enable non-intrusive structural analysis.
